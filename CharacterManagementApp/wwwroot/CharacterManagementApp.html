Dungeons & Dragons 5e Character Management App

  • New Character: Fill out and submit the character creation form to add a new character to the database.
  • View Character: Select an existing character from the drop down menu to view its information.
  • Update Character: Choose a character and create a custom form for updating specific details about it.
  • Inventory: View a character's inventory and add or remove items.
  • Spells: View a character's spell list and add or remove spells.
